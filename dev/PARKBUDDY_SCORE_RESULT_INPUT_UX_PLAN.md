# ParkBuddy 스코어 입력/결과 확인 UX 개선 계획

## 작업명

Polish score and result input experience

## 목적

운영자가 모바일에서 스코어 입력 현황을 빠르게 파악하고, 누락된 참가자를 바로 찾아 입력할 수 있도록 스코어 입력/결과 화면을 보강한다.

## 적용 화면

- /admin/rounds/[id]/scores
- /admin/rounds/[id]/results

## 개선 방향

1. 스코어 입력 화면 상단에 입력 완료율, 완료 인원, 미입력 인원을 표시한다.
2. 참가자별 입력 카드에 입력 완료/미입력 배지를 추가한다.
3. 숫자 입력에는 모바일 숫자 키패드가 뜨도록 inputMode를 보강한다.
4. 저장 버튼에 현재 완료 인원을 표시해 저장 직전 상태를 확인할 수 있게 한다.
5. 결과 화면에는 현재 1위, 입력 완료율, Top 3 요약을 추가한다.
6. 결과 목록에서는 미입력 인원을 시각적으로 구분한다.

## 검증 기준

- npm run verify 통과
- 모바일에서 스코어 입력 현황이 첫 화면에 보여야 한다.
- 미입력 참가자를 눈으로 쉽게 찾을 수 있어야 한다.
- 숫자 입력 시 모바일 숫자 키패드가 떠야 한다.
- 결과 화면에서 현재 1위와 Top 3가 바로 보여야 한다.
- 미입력 인원이 결과 화면에서 명확히 구분되어야 한다.
