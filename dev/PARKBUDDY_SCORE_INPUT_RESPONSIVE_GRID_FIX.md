# ParkBuddy 스코어 입력 반응형 그리드 보정

## 작업 목적

스코어 입력 화면에서 태블릿/데스크탑 폭에 맞춰 요약 카드와 회원 입력 카드가 자연스럽게 확장되도록 보정한다.

## 변경 범위

- `src/components/admin/round-score-input-form.tsx`
- `dev/PARKBUDDY_OPERATOR_DEV_STATUS.md`

## 반영 내용

1. 회원 스코어 카드를 태블릿/데스크탑 폭에서 2열 또는 3열 그리드로 표시한다.
2. 총 타수와 스테이블포드 포인트 입력창은 카드 폭에 맞춰 자연스럽게 줄어들거나 늘어나도록 `minmax` 기반 그리드를 유지한다.
3. 모바일이 아닌 화면에서도 일자/참가/입력 요약 카드가 보이도록 `sm:hidden` 제한을 제거했다.
4. 모바일에서는 미입력만 보기 버튼이 요약 카드 아래 전체 폭으로 표시된다.
5. 태블릿/데스크탑에서는 미입력만 보기 버튼이 입력 카드 오른쪽에 배치된다.
6. 기존 스코어 저장 로직, 미입력 필터 로직, 숨김 input 보존 로직은 변경하지 않았다.

## 확인 항목

- 모바일: 요약 카드 3개 + 미입력만 보기 버튼이 아래에 표시되는지 확인
- 태블릿/데스크탑: 일자/참가/입력/미입력만 보기 버튼이 한 줄에 표시되는지 확인
- 태블릿/데스크탑: 회원 카드가 화면 폭에 따라 2열 또는 3열로 표시되는지 확인
- 입력값 저장 및 미입력 필터 동작이 기존과 동일한지 확인
