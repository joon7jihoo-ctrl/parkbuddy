# ParkBuddy 회원 검색/필터 카드 보정 v2

## 목표

회원 관리 검색을 서버 URL 쿼리 의존 방식에서 클라이언트 실시간 필터링 방식으로 보강한다.

## 반영 내용

- 회원 관리 상단 필터 카드는 라운드 관리처럼 모바일에서도 4개가 한 줄에 작게 보이도록 수정했다.
- 검색창 입력 중간 상태인 `ㅎ`, `ㅎㅗ`, `호`도 즉시 필터링되도록 했다.
- 전화번호 검색은 `7` 한 글자만 입력해도 연락처 숫자에 7이 포함된 회원을 즉시 보여준다.
- 필터 카드를 누르면 검색어를 비우고 해당 조건 전체 목록을 보여준다.
- 기존 `useEffect` 안의 동기 setState 경고를 제거했다.
