# ParkBuddy Mypage Score Graph and Scores Menu Fix

## 목표
- 마이페이지의 내 연락처는 전화 연결 링크가 아니라 단순 정보로 표시한다.
- 라운딩 기록 화면에서 보던 스코어 추이 그래프를 마이페이지에도 제공한다.
- 일반 기록 조회 화면에서 운영자 성격의 라운딩 등록 메뉴를 제거한다.

## 변경 내용
1. `/mypage`
   - 연락처 tel 링크 제거
   - 하이픈 포함 연락처 텍스트 유지
   - 최근 20개 스코어 추이 그래프 추가
2. `/scores`
   - 상단 `라운딩 등록` 액션 제거

## 확인 포인트
- 마이페이지 연락처가 클릭 링크처럼 보이지 않는지
- 마이페이지에 최근 스코어 추이 그래프가 표시되는지
- 스코어 기록 화면 상단에 라운딩 등록 버튼이 없는지
