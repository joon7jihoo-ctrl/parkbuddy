# ParkBuddy Public Member Card Popup Adjust

## 목표
- 회원 목록의 회원명 글자 크기를 기존 수준으로 낮춘다.
- 연락처/핸디캡/가입일 카드가 회원명 아래로 길게 쌓이지 않고, 회원명 오른쪽 영역에 2줄 구조로 보이도록 정리한다.
- 회원 상세 팝업은 화면 하단 바텀시트가 아니라 화면 중앙 모달로 표시한다.
- 팝업 하단 닫기 버튼은 제거하고, 우측 상단 닫기 버튼만 유지한다.

## 확인 항목
- `/members`에서 회원명이 과하게 크지 않은지 확인한다.
- 회원 카드가 왼쪽 이름/역할, 오른쪽 상세 정보 구조로 보이는지 확인한다.
- 이름 클릭 시 팝업이 화면 중앙에 뜨는지 확인한다.
- 팝업 하단 닫기 버튼이 없는지 확인한다.
