2026-06-13

## 진행 내용

입력된 라운드 스코어를 기반으로 결과/순위 보기 페이지를 추가했다.

## 변경 파일

```txt
src/app/(app)/admin/rounds/[id]/results/page.tsx
dev/2026-06-13-round-results-page-report.md
```

## 구현 기능

```txt
라운드별 결과 페이지
경기 형태 표시
점수 계산 방식 표시
스트로크 플레이 총타수 기준 정렬
스테이블포드 포인트 기준 정렬
신페리오/매치 플레이는 임시 총타수 기준 안내
```

## 보안 평가

결과 페이지는 운영진 권한 확인을 통과해야 접근할 수 있다.

라운드 조회 시 현재 운영진의 club_id를 기준으로 다시 제한한다.

다른 동호회의 라운드 ID를 URL에 직접 입력해도 조회되지 않는다.
