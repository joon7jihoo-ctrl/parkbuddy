# ParkBuddy Score Input Auto Collapse Fix

## 목적

스코어 입력 화면에서 완료자 요약 카드의 불필요한 안내 문구를 제거하고, 완료자를 펼친 상태가 미입력 필터에 영향을 주지 않도록 수정한다.

## 수정 범위

- `src/components/admin/round-score-input-form.tsx`

## 반영 내용

1. 완료자 요약 카드의 `펼치기` 문구 삭제
2. 완료자 요약 카드를 선택하면 해당 회원만 입력 카드로 펼침
3. 펼쳐진 완료자 카드에서 입력창/라벨 외 영역을 누르면 다시 접힘
4. `미입력만 보기`를 켤 때 펼친 완료자를 자동으로 접음
5. 미입력 필터는 오직 실제 입력값 유무만 기준으로 판단하도록 수정
6. 접힌 완료자는 hidden input으로 값을 계속 보존

## 화면 확인 포인트

1. 완료자 요약 카드에 `펼치기` 문구가 보이지 않는지 확인
2. 완료자 요약 카드를 누르면 입력창이 펼쳐지는지 확인
3. 펼쳐진 카드에서 입력창 외 영역을 누르면 다시 접히는지 확인
4. 완료자를 펼친 상태에서 `미입력만 보기`를 눌러도 완료자가 미입력 목록에 남지 않는지 확인
5. 모든 참가자 입력 완료 상태에서 `미입력만 보기`를 누르면 완료 안내가 정상 표시되는지 확인
