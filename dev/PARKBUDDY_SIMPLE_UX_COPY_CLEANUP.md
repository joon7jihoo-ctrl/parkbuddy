# ParkBuddy Simple UX Copy Cleanup

## 목적

화면에서 꼭 필요한 정보만 남기고, 사용자가 이미 행동으로 알 수 있는 안내 문구를 제거한다.

## 적용 기준

- `ParkBuddy` 외 영어 노출은 계속 줄인다.
- 클릭 가능한 카드에 `자세히 보기`처럼 중복 행동 문구를 두지 않는다.
- 같은 화면/기능은 같은 이름으로 부른다.
- 설명은 기능 이해에 꼭 필요한 경우에만 남긴다.

## 변경 내용

### 홈

- 홈 상단의 `하단에는 홈 버튼만 두고...` 안내 문구를 제거했다.
- 주요 메뉴 카드의 반복 설명을 제거하고 제목 중심으로 정리했다.
- `최근 공지` 표현을 `게시판` 기준으로 정리했다.
- 게시판 카드의 빈 상태 문구를 `게시글이 없습니다.`로 통일했다.

### 게시판

- 게시판 상단 설명 문구를 제거했다.
- 게시글 카드를 누르면 상세 화면으로 이동하므로 `자세히 보기` 문구를 제거했다.
- 빈 상태 설명을 줄이고 `게시글이 없습니다`만 남겼다.

### 글쓰기

- 상단 설명 문구를 제거했다.
- 게시판 유형 옵션을 `자유글`, `공지`로 간단히 정리했다.
