# ParkBuddy MyPage Upcoming Schedule Summary

## Date
2026-06-16

## Scope
- `src/app/(app)/mypage/page.tsx`

## Goal
마이페이지에서 내 기본 정보와 기록만 확인하는 흐름을 넘어, 다가오는 일정과 내 참석 선택 상태를 빠르게 확인할 수 있도록 모바일 Dense UX를 보강한다.

## Changes
- 마이페이지에 `다가오는 내 일정` 섹션을 추가했다.
- 예정 일정 최대 3건을 표시한다.
- 각 일정 카드에 일정 유형, 제목, 일시, 장소, 내 참석 상태를 표시한다.
- 일정 유형은 `/schedule`과 동일하게 `정기 라운딩`, `대회`, `번개` 기준으로 표시한다.
- 내 선택 상태는 `참석`, `불참`, `미선택` 배지로 표시한다.
- 참석 수와 전체 투표 수를 표시하고, 참석 비율을 프로젝트 색상 계열의 미니 가로 막대로 표현한다.
- 카드와 `일정 보기` 버튼은 `/schedule`로 연결해 실제 투표 화면으로 바로 이동할 수 있게 했다.

## Non-goals
- Supabase DB 구조 변경 없음.
- 일정 투표 저장 로직 변경 없음.
- `/schedule` 투표 UI 변경 없음.
- 게시판/회원 검색/스코어 입력 화면 변경 없음.

## Verification
- `npm run verify` 통과.

## Manual QA
1. `/mypage` 접속.
2. `다가오는 내 일정` 섹션이 보이는지 확인.
3. 예정 일정이 있을 때 최대 3건이 compact 카드로 표시되는지 확인.
4. 각 카드의 내 상태가 참석/불참/미선택으로 표시되는지 확인.
5. 카드 또는 `일정 보기`를 누르면 `/schedule`로 이동하는지 확인.
6. 모바일에서 카드 높이와 여백이 과도하지 않은지 확인.
7. 태블릿/데스크탑에서 일정 카드가 3열로 자연스럽게 배치되는지 확인.
