# ParkBuddy 일반 회원 목록/상세 모바일 레이아웃 개선 계획

## 작업 목적

운영자 화면 중심으로 정리된 모바일 UI 패턴을 일반 회원이 보는 회원 목록과 회원 상세 화면에도 적용한다.

## 수정 대상

- src/app/(app)/members/page.tsx
- src/app/(app)/members/[id]/page.tsx

## 개발 방향

### 회원 목록

- 일반 회원 조회 화면으로 단순화한다.
- 운영자 전용 관리/등록/연결 상태 정보는 노출하지 않는다.
- 회원 이름을 누르면 상세 화면으로 이동한다.
- 연락처 번호를 누르면 전화 연결이 가능하게 유지한다.
- 모바일에서는 카드 안의 정보를 2열로 압축해 스크롤을 줄인다.

### 회원 상세

- 이름, 역할, 연락처, 핸디캡, 가입일을 카드형으로 정리한다.
- 연락처는 한국 전화번호 형식으로 표시하고 번호 선택 시 전화 연결이 가능하게 한다.
- 라운딩 수, 평균, 베스트 기록을 3개 요약 카드로 표시한다.
- 하단에는 회원 목록으로 돌아가는 CTA만 둔다.

## 검증 기준

- npm run verify 통과
- /members 화면에서 운영자 전용 정보가 보이지 않을 것
- /members/[id] 화면에서 연락처 클릭 시 전화 연결될 것
- 모바일에서 카드가 과하게 길어지지 않을 것
- 하단 CTA가 홈 버튼과 겹치지 않을 것
