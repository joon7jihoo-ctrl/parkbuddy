# ParkBuddy 홈/네비게이션 허브 정리 계획

## 작업명

Remove admin dashboard and simplify bottom navigation

## 목적

하단 네비게이션을 홈 버튼 하나로 단순화하고, 홈 화면을 주요 기능 허브로 재구성한다.

## 반영 내용

- `/admin` 대시보드 페이지는 별도 화면으로 유지하지 않고 `/` 홈으로 이동한다.
- 하단 네비게이션은 가운데 홈 버튼 하나만 표시한다.
- 홈 화면에서 `회원 목록` 카드는 로그인 사용자의 역할에 따라 이동한다.
  - 운영자: `/admin/members`
  - 일반 회원: `/members`
- 운영자 홈에는 `라운딩 관리`, `작업 관리` 카드를 추가한다.
- 일반 회원 홈에는 일정, 스코어, 게시판, 마이페이지 이동 카드를 유지한다.
- 다가오는 일정과 최근 공지는 홈 화면에 압축해서 유지한다.

## 검증 기준

- `npm run verify` 통과
- 하단 네비게이션에 홈 버튼만 표시
- `/admin` 접근 시 `/`로 이동
- 운영자 홈에서 회원 목록 클릭 시 회원 관리로 이동
- 일반 회원 홈에서 회원 목록 클릭 시 일반 회원 목록으로 이동
- 운영자 홈에서 라운딩 관리, 작업 관리 카드가 정상 이동
