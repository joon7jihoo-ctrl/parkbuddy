# 라운드 참가자 선택 검색/선택 기능 안정화 보고서

## 작성일

2026-06-13

## 작업 목적

라운드 참가자 선택 화면에서 회원 검색, 선택된 회원만 보기, 전체 선택/전체 선택 해제 토글, 선택 인원 표시가 정상적으로 동작하지 않는 문제를 수정했다.

## 수정 범위

- src/components/admin/member-search-filter.tsx

## 수정 내용

- 검색창 입력 시 회원 목록이 즉시 필터링되도록 수정
- 선택된 회원만 보기 버튼이 실제 체크 상태를 기준으로 동작하도록 수정
- 전체 선택 버튼이 현재 표시 중인 회원 기준으로 전체 선택/전체 선택 해제를 토글하도록 수정
- 검색창 아래 전체 회원 수와 선택 회원 수가 실제 체크박스 상태와 일치하도록 수정
- 활성 회원 영역의 현재 선택 인원 문구가 저장 버튼을 누르기 전에도 즉시 갱신되도록 수정
- React 상태 대신 DOM 동기화 방식으로 처리하여 hooks lint 오류를 피함

## 확인 항목

- npm run verify 통과
- 회원 검색 동작 확인
- 선택된 회원만 보기 동작 확인
- 전체 선택/전체 선택 해제 토글 확인
- 활성 회원의 현재 선택 인원 숫자 즉시 반영 확인
