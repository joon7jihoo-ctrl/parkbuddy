# ParkBuddy 회원/라운드 목록 필터 및 CTA 정리 계획

## 목적

회원 관리와 라운드 목록의 상단 필터 카드를 실제 목록 필터로 동작하게 만들고, 일반 회원 목록에서는 운영자 전용 정보와 버튼을 제거한다.

## 반영 내용

- `/admin/members` 검색을 `GET` 쿼리 기반으로 명확히 정리했다.
- 회원 관리 상단 카드(`활성 전체`, `연결 완료`, `연결 대기`, `코드 필요`)를 선택하면 해당 조건의 회원만 보이도록 했다.
- 회원 관리의 일반 회원 목록 이동 버튼과 최근 관리자 작업 카드를 제거했다.
- 회원 관리 하단 고정 CTA를 `대시보드`, `회원 등록`으로 맞췄다.
- 일반 회원 목록 `/members`에서 운영자 전용 관리/등록 버튼과 연결 상태/코드 만료 정보를 제거했다.
- `/admin/rounds`에서 상태별 보기 버튼을 제거하고, 상단 상태 카드(`전체`, `예정`, `완료`, `취소`)를 통해 같은 화면에서 라운드를 필터링하도록 했다.
- 라운드 목록 하단 고정 CTA를 `대시보드`, `라운드 생성`으로 맞췄다.

## 검증 기준

- `/admin/members`에서 이름/전화번호 검색이 동작한다.
- 회원 관리 상단 카드를 누르면 제목에 맞는 회원만 보인다.
- `/members`에서 운영자 전용 관리/등록/연결 상태 정보가 보이지 않는다.
- `/admin/rounds`에서 상태별 보기 버튼이 사라지고 상태 카드 필터가 동작한다.
- `/admin/rounds` 하단에 대시보드/라운드 생성 고정 CTA가 보인다.
- `npm run verify`가 통과한다.
