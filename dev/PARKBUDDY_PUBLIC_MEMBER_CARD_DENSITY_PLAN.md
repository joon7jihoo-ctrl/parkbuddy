# ParkBuddy Public Member Card Density Plan

## 목표
- 일반 회원 목록 카드의 내부 여백을 줄여 한 화면에 더 많은 회원 정보가 보이도록 개선한다.
- 회원 이름, 역할, 가입일, 핸디캡, 연락처의 정보 우선순위는 유지한다.
- 연락처는 하단 전체 폭을 유지해 한 줄로 읽히도록 한다.

## 변경 내용
- 회원 카드 padding과 카드 내부 gap 축소
- 회원명 글자 크기는 과하지 않게 유지
- 정보 셀의 padding과 라운드 크기 축소
- 목록 카드 간격 축소
- 상단 요약 카드 높이도 소폭 축소

## 확인 포인트
- 모바일에서 한 화면에 더 많은 회원 카드가 보이는지
- 연락처가 여전히 한 줄로 보이는지
- 회원 이름 클릭 팝업과 전화 연결 기능이 유지되는지
