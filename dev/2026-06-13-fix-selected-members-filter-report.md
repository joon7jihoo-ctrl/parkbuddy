# 라운드 참가자 선택 화면 선택 회원 보기 오류 수정

## 작성일

2026-06-13

## 작업 내용

라운드 참가자 선택 화면의 회원 검색 컴포넌트에서 선택된 회원만 보기 기능이 정상적으로 동작하도록 보정했다.

## 수정 파일

- src/components/admin/member-search-filter.tsx

## 주요 수정 사항

- 체크박스 변경 시 선택 인원과 표시 목록을 즉시 다시 계산하도록 수정
- 선택된 회원만 보기 상태에서 체크 해제된 회원은 즉시 숨겨지도록 수정
- 검색어와 선택 상태가 함께 적용되도록 필터 기준 정리
- 전체 선택 버튼은 현재 표시 중인 회원 기준으로 동작하도록 수정
- 전체 선택 해제 버튼은 선택 상태에 따라 자동으로 표시 문구가 바뀌도록 유지

## 확인 항목

- 참가자 선택 화면에서 회원 검색이 정상 동작한다.
- 선택된 회원만 보기 버튼이 정상 동작한다.
- 선택된 회원만 보기 상태에서 체크 해제하면 해당 회원이 목록에서 사라진다.
- 전체 회원 보기 버튼을 누르면 전체 목록으로 돌아온다.
- 전체 선택과 전체 선택 해제가 정상 동작한다.
- npm run verify가 통과한다.
