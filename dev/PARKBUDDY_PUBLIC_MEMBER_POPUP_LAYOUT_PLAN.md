# ParkBuddy Public Member Popup Layout Plan

## 목표
- 일반 회원 목록 상단 요약 카드의 높이를 줄여 한 화면 정보 밀도를 높인다.
- 회원 상세는 별도 페이지가 아니라 팝업으로 열어 목록 흐름을 유지한다.
- 회원 카드 우측 원형 이니셜 UI는 제거하고, 카드 구조를 더 간결하게 정리한다.

## 이번 변경
1. `/members`
   - 상단 요약 카드(전체/운영진/회원) 높이 축소
   - 회원 이름 클릭 시 팝업 상세 표시
   - 연락처/핸디캡/가입일을 2열 카드 구조로 정리
   - 원형 이니셜 UI 제거
2. `/members/[id]`
   - 팝업 전환으로 불필요해져 라우트 제거
3. 데이터
   - 목록 화면에서 회원 통계 데이터를 함께 조회해 팝업 상세에 재사용

## 확인 포인트
- 모바일에서 상단 3개 요약 카드가 이전보다 낮고 촘촘하게 보이는지
- 회원 이름 클릭 시 별도 페이지 이동 없이 팝업이 열리는지
- 팝업에서 연락처/핸디캡/가입일/통계가 정상 표시되는지
- 연락처 클릭 시 전화 연결이 동작하는지
