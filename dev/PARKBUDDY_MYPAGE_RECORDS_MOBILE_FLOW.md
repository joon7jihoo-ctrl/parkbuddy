# ParkBuddy MyPage Records Mobile Flow

## 목표
- 일반 회원이 본인 정보와 최근 라운딩 기록을 한 화면에서 쉽게 확인하도록 마이페이지를 정리한다.
- MVP 요구사항 중 `내 최근 기록 및 통계` 흐름을 모바일 우선으로 보강한다.

## 변경 내용
1. `/mypage`
   - 기본 회원 정보 카드 정리
   - 연락처 하이픈 표시 및 전화 연결 유지
   - 라운딩/평균/베스트 요약 카드 추가
   - 최근 기록 목록 추가
   - `/scores` 전체 기록 이동 링크 추가
2. 데이터
   - `member_score_stats`에서 내 통계 조회
   - `member_round_totals`에서 최근 기록 5개 조회

## 확인 포인트
- 모바일에서 기본 정보와 통계가 한눈에 들어오는지
- 연락처가 `010-1234-5678` 형태로 보이는지
- 최근 기록 클릭 시 스코어 상세로 이동하는지
- 스코어가 없는 회원에게 빈 상태 문구가 자연스럽게 보이는지
