# ParkBuddy Board Notice Mobile Flow

## 목표
- 일반 회원 MVP의 공지사항/게시판 조회 흐름을 모바일에서 더 읽기 쉽게 정리한다.
- 공지, 고정, 자유게시글을 카드 안에서 명확히 구분한다.
- 글쓰기 화면은 모바일에서도 등록/취소 동작이 명확하도록 정리한다.

## 변경 내용
1. `/board`
   - 전체/공지/고정 요약 카드 추가
   - 게시글 카드를 더 촘촘한 모바일 카드로 정리
   - 공지/자유/고정 배지 정리
2. `/board/[id]`
   - 게시글 상세 본문을 읽기 쉬운 카드로 정리
   - 게시판으로 돌아가기 버튼 추가
3. `/board/new`
   - 등록/취소 버튼을 하단 2열로 정리
   - 안내 문구를 카드형으로 정리

## 확인 포인트
- 모바일에서 게시글 목록이 한눈에 들어오는지
- 공지와 고정 게시글이 쉽게 구분되는지
- 상세 화면 본문 가독성이 좋은지
- 글쓰기 화면에서 등록/취소 버튼이 명확한지
