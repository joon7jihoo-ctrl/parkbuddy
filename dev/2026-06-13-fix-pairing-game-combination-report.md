# 경기 방식 조합 저장 오류 수정 보고서

## 작업 내용

- 조 편성 화면과 저장 로직의 매치 플레이 값 기준을 `match`로 통일했습니다.
- 기존 `match_play` 값이 들어와도 저장 로직에서 `match`로 보정되도록 했습니다.
- `team` 값이 들어와도 `team_match`로 보정되도록 했습니다.
- 조 편성 화면에서 경기 형태 값이 예상 목록과 달라도 기본 옵션으로 안전하게 처리하도록 했습니다.

## 확인 항목

- 조 편성 화면에서 청백전 + 스트로크 플레이 저장
- 개인전 + 신페리오 저장
- 포볼 + 스테이블포드 저장
- 매치 플레이 조합 저장
