# ParkBuddy Schedule Attendance Mobile Flow

## 목표
- 일반 회원이 `/schedule`에서 라운딩 일정을 빠르게 확인하고 참석/미정/불참을 선택할 수 있게 한다.
- 모바일 화면에서 일정 카드의 정보 밀도를 높이고, 내 선택 상태를 즉시 확인할 수 있게 한다.

## 변경 내용
1. 일정 목록 카드
   - 일정 유형과 내 선택 상태를 상단 배지로 표시한다.
   - 참석 인원은 우측 강조 카드로 표시한다.
   - 일시/장소를 2열 정보 카드로 정리한다.
   - 참석/미정/불참 집계를 작은 상태 카드로 표시한다.
2. 참석 투표 버튼
   - 현재 사용자의 선택 상태를 버튼 스타일로 강조한다.
   - 버튼은 모바일에서도 3열 유지한다.
   - 참석/미정/불참 선택 흐름은 기존 서버 액션을 그대로 사용한다.

## 확인 포인트
- 내 선택 배지가 미선택/참석/미정/불참으로 정상 표시되는지
- 참석 버튼을 누르면 새로고침 후 선택 상태가 강조되는지
- 모바일에서 일정 카드가 과하게 길지 않은지
- 운영자만 일정 등록 버튼을 볼 수 있는지
