# ParkBuddy 스코어 입력 완료자 접기 UX 개선

작성일: 2026-06-15

## 목적

스코어 입력 화면에서 이미 입력 완료된 참가자가 많을 때 모바일 스크롤 부담을 줄이기 위해 완료자를 간단한 요약 카드로 접을 수 있는 UX를 추가한다.

## 변경 범위

- `src/components/admin/round-score-input-form.tsx`
- `dev/PARKBUDDY_OPERATOR_DEV_STATUS.md`

## 반영 내용

- 완료 스코어가 1명 이상 있고 `미입력만 보기`가 아닌 상태에서 `완료자 접기` 버튼 표시
- 완료자 접기 상태에서는 입력 완료 참가자를 요약 카드로 표시
- 요약 카드에는 회원명, 핸디캡, 총 타수, 포인트를 표시
- 요약 카드의 `수정` 버튼을 누르면 해당 참가자만 다시 입력 카드로 펼쳐짐
- 접힌 완료자의 값은 hidden input으로 보존되어 저장 시 유실되지 않음
- 미입력 참가자는 항상 기존 입력 카드로 유지

## 화면 확인 항목

1. 스코어 입력 화면에서 완료자가 있을 때 `완료자 접기` 버튼이 보이는지 확인
2. 버튼을 누르면 입력 완료 참가자만 요약 카드로 줄어드는지 확인
3. 미입력 참가자는 계속 입력 가능한 카드로 남는지 확인
4. 요약 카드의 `수정` 버튼을 누르면 해당 참가자 입력창이 다시 나타나는지 확인
5. 접힌 상태에서 저장해도 기존 입력값이 유지되는지 확인
