# ParkBuddy 결과 화면 중복 정보 축소 및 하단 CTA 정리

## 작업 목적

결과 화면에서 모바일 스크롤을 줄이고, 중복되는 공유/입력 현황 카드를 정리한다.

## 반영 내용

- 상단의 결과 링크 복사, 결과 요약 복사, 인쇄용 결과표 버튼 제거
- 별도 카카오톡 공유용 요약 카드 제거
- 결과 요약 카드 안에 접기/펼치기 공유/인쇄 옵션 추가
- 펼쳤을 때 요약 복사와 인쇄용 결과표 버튼 제공
- 결과 입력 현황 카드 제거
- 미입력 스코어 안내 카드 제거
- 결과 순위 안의 미입력 표시만 유지
- 라운드 목록, 스코어 입력 버튼을 화면 하단 고정 CTA로 정리

## 검증 항목

- /admin/rounds/[id]/results 화면에서 중복 카드가 사라졌는지 확인
- 결과 요약 카드에서 공유/인쇄 옵션을 펼칠 수 있는지 확인
- 요약 복사와 인쇄용 결과표 버튼이 펼친 카드 내부에 있는지 확인
- 라운드 목록, 스코어 입력 버튼이 하단 고정 CTA로 보이는지 확인
- 모바일에서 스크롤량이 줄었는지 확인
- npm run verify 통과 확인
