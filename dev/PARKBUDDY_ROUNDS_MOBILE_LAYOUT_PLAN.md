# ParkBuddy 라운드 관리 모바일 우선 레이아웃 개선 계획

## 작업명

Polish round management mobile layout

## 배경

라운드 관리 화면은 모바일과 태블릿 폭에서는 카드 내용이 비교적 잘 보이지만, 데스크탑 폭에서는 넓은 화면을 충분히 사용하지 못하고 라운드 정보가 왼쪽 좁은 영역에 눌려 보이는 문제가 있었다.

또한 모바일에서는 버튼과 상세 정보가 계속 세로로 쌓여 스크롤이 길어지는 문제가 있었다. 라운드 운영 화면은 현장에서 모바일로 빠르게 확인하는 경우가 많으므로, 스크롤을 줄이면서도 핵심 정보가 충분히 보이도록 재배치가 필요하다.

## 수정 방향

1. 라운드 관리 화면의 최대 폭을 넓혀 데스크탑에서도 카드 정보가 잘 보이게 한다.
2. 라운드 기본 정보는 모바일에서도 2열 카드형 정보 그리드로 배치한다.
3. 핵심 작업 버튼은 참가자, 조 편성, 스코어, 결과만 우선 노출한다.
4. 수정, 복제, 상태 변경은 `더보기 · 상태 관리` 접기/펼치기 영역으로 이동한다.
5. 삭제는 `위험 작업` 접기/펼치기 영역으로 분리한다.
6. 메모는 필요한 경우에만 펼쳐서 볼 수 있게 한다.
7. 데스크탑에서는 핵심 작업 버튼 영역을 카드 오른쪽 dock처럼 배치해 넓은 화면을 자연스럽게 사용한다.

## 검증 기준

- `npm run verify` 통과
- `/admin/rounds` 데스크탑 폭에서 라운드 정보가 좁게 눌리지 않음
- `/admin/rounds` 모바일 폭에서 기본 정보가 2열 중심으로 표시됨
- 모바일에서 버튼이 무조건 세로로 길게 쌓이지 않음
- 수정/복제/상태 변경은 접기/펼치기 안에 있음
- 삭제는 위험 작업 접기/펼치기 안에 있음
- 삭제된 라운드 목록과 복구 흐름은 기존 기능을 유지함
