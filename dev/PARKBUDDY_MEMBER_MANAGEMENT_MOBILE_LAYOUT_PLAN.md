# ParkBuddy 회원 관리 모바일 레이아웃 개선 계획

## 작업명

Polish member management mobile layout

## 목표

라운드 관리 화면에 적용한 모바일 우선 카드 패턴을 회원 관리 화면에도 확장한다.

## 반영 내용

- 관리자 회원 관리 화면을 `max-w-7xl` 기반 카드 레이아웃으로 확장
- 회원 검색/상태 필터/핵심 현황을 상단 압축 카드로 정리
- 회원 카드는 모바일에서 2열 정보 그리드와 접기/펼치기 관리 작업을 사용
- 데스크탑에서는 회원 정보와 관리 버튼이 좌우로 자연스럽게 배치
- 최근 관리자 작업은 접기/펼치기로 이동해 초기 스크롤을 줄임
- 회원 등록/수정 화면은 2열 입력 폼과 하단 고정 저장 CTA 적용

## 화면 확인 기준

- `/admin/members` 모바일에서 검색/필터/회원 카드가 과하게 길지 않아야 한다.
- 회원별 수정/코드 재발급/비활성화/복구 작업은 모바일에서 접기/펼치기로 확인한다.
- `/admin/members/new`, `/admin/members/[id]/edit` 저장 버튼은 모바일 하단에 고정된다.
- 데스크탑에서는 회원 카드가 넓은 화면을 자연스럽게 사용해야 한다.
