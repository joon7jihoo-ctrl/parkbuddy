# 라운드 참가자 회원 검색/선택 기능 재안정화

## 작업 일자

2026-06-13

## 작업 목적

라운드 참가자 선택 화면의 회원 검색, 선택된 회원만 보기, 전체 선택/해제 토글, 선택 인원 즉시 표시 기능을 다시 안정화했다.

## 변경 파일

- src/components/admin/member-search-filter.tsx
- dev/2026-06-13-restabilize-member-search-filter-report.md

## 수정 내용

- 검색 입력 시 현재 회원 목록을 즉시 필터링하도록 수정
- 선택된 회원만 보기 토글 동작을 보정
- 전체 선택/전체 선택 해제를 현재 표시 목록 기준으로 동작하도록 수정
- 검색창 아래 전체 회원 수와 선택 회원 수를 실제 체크박스 상태 기준으로 갱신
- 활성 회원 설명 영역의 현재 선택 인원도 즉시 갱신
- React 상태 의존을 줄이고 실제 체크박스 DOM 상태를 기준으로 계산하도록 안정화

## 확인 항목

- npm run verify 통과
- 참가자 선택 화면 접속
- 검색창 동작 확인
- 선택된 회원만 보기 동작 확인
- 전체 선택/전체 선택 해제 토글 확인
- 참가자 저장 기능 확인
