# ParkBuddy 개발보고서 - 라운드 목록 스코어 입력 버튼 추가

## 작성일

2026-06-13

## 진행 내용

라운드 목록에서 라운드 ID를 직접 확인하지 않아도 스코어 입력 화면으로 이동할 수 있도록 버튼을 추가했다.

## 변경 파일

```txt
src/app/(app)/admin/rounds/page.tsx
dev/2026-06-13-add-round-score-button-report.md
```

## 구현 기능

각 라운드 카드에 다음 버튼을 표시한다.

```txt
참가자 선택
조 편성
스코어 입력
```

## 보안 평가

이번 변경은 이동 버튼 추가와 목록 표시 보정이다.

스코어 저장 권한 검증은 기존처럼 스코어 저장 RPC와 서버 액션에서 수행한다.

## 완료 기준

```txt
1. npm run verify 통과
2. /admin/rounds 접속 가능
3. 각 라운드 카드에 스코어 입력 버튼 표시
4. 스코어 입력 버튼 클릭 시 /admin/rounds/[id]/scores 이동
```
