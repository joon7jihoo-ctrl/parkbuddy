# 라운드 참가자 선택 UX 및 조 편성 오류 수정 보고서

## 작업 목적

라운드 참가자 선택 화면의 검색 영역 사용성을 개선하고, 라운드 복제 후 조 편성 화면에서 경기 형태 값이 예상 목록과 다를 때 발생하던 런타임 오류를 수정했다.

## 수정 내용

- 회원 검색 영역의 위아래 여백과 입력창 높이를 넓힘
- 선택 인원/전체 인원 표시 유지
- 선택 버튼을 전체 선택/전체 선택 해제로 토글되도록 수정
- 선택된 회원만 보기 기능 유지
- 조 편성 폼에서 경기 형태 값이 비정상이어도 기본값으로 보정
- 점수 계산 옵션이 undefined가 되지 않도록 방어 로직 추가

## 변경 파일

- src/components/admin/member-search-filter.tsx
- src/components/admin/round-pairing-form.tsx

## 확인 항목

- npm run verify 통과
- 참가자 선택 화면에서 검색창 여백이 충분한지 확인
- 전체 선택/전체 선택 해제 토글 동작 확인
- 복제한 라운드에서 조 편성 화면 진입 시 오류가 발생하지 않는지 확인
