# 조 편성 및 라운드 목록 점수 계산 방식 수정 작업 커밋 보고서

## 작성일
2026-06-13

## 목적
조 편성 저장 이후 라운드 목록에서 점수 계산 방식이 미지정으로 보이는 문제를 수정한 작업을 Git에 안전하게 저장한다.

## 포함 대상
- 조 편성 화면 추천 경기 방식 초기값 적용
- 조 편성 저장 시 경기 형태와 점수 계산 방식 저장 보정
- 라운드 목록에서 점수 계산 방식 한글 표시 보정
- 관련 개발 보고서

## 실행 전제
- npm run verify가 통과해야 커밋을 진행한다.
- 현재 브랜치에 변경사항이 있을 때만 커밋한다.
- 원격 저장소가 연결되어 있으면 현재 브랜치로 push한다.
