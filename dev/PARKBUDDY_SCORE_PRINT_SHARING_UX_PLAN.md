# ParkBuddy 스코어 인쇄/공유 UX 개선 계획

## 작업명

Polish score print and sharing experience

## 목표

라운드 결과를 운영자가 확인한 뒤 카카오톡으로 공유하거나 인쇄/PDF 저장할 때 더 보기 쉽고 실수 없이 전달할 수 있도록 개선한다.

## 변경 방향

- 결과 화면에 카카오톡 공유용 요약 문구를 추가한다.
- 현재 1위, Top 3, 입력 완료율, 미입력 인원을 공유 문구에 포함한다.
- 결과 링크 복사와 별도로 결과 요약 복사 버튼을 제공한다.
- 인쇄용 결과표에도 현재 1위, Top 3, 입력 완료율을 표시한다.
- 인쇄 화면은 불필요한 앱 네비게이션 없이 결과표 중심으로 유지한다.

## 검증 항목

- `/admin/rounds/[id]/results`에서 결과 요약 복사 버튼이 보인다.
- 복사한 문구에 라운드명, 골프장, 날짜, 1위, Top 3, 입력 완료율이 포함된다.
- `/admin/rounds/[id]/results/print`에서 1위와 Top 3이 인쇄 미리보기에도 보인다.
- `npm run verify`가 통과한다.
