# ParkBuddy 회원 검색/필터 카드 보정

## 목표

회원 관리 검색을 실시간 URL 쿼리 기반으로 정리하고, 한글 초성/자소/부분 일치/스마트 다이얼/전화번호 부분 검색을 안정화한다.

## 반영 내용

- 검색창 입력 중에도 URL 쿼리를 갱신해 목록이 즉시 바뀌도록 했다.
- `ㅎ`, `ㅎㅗ`, `호`, `홍`, `7`처럼 입력 중간 상태와 한 글자 숫자도 검색 대상으로 처리한다.
- 회원 관리 상단 필터 카드는 라운드 관리 카드처럼 클릭 시 검색어를 무시하고 조건별 전체 목록을 보여준다.
- 연락처 표기는 `010-1234-5678` 형식으로 통일하고, 번호를 누르면 전화 연결이 가능하게 했다.
- 일반 회원 목록은 운영자 전용 정보와 버튼을 제거하고 조회 중심으로 단순화했다.
- 하단 네비게이션은 운영자/일반 회원에 따라 회원 메뉴 목적지를 분리한다.
