# PARKBUDDY_SCORE_RESULT_SIMPLE_UX_CLEANUP

## 목적

스코어/결과 화면에서 반복 안내 문구를 줄이고, 모바일에서 핵심 정보가 먼저 보이도록 카드 밀도를 정리한다.

## 변경 범위

- `/scores`
- `/scores/[roundId]`
- `/mypage` 최근 기록 영역
- `/admin/rounds/[id]/scores`
- `/admin/rounds/[id]/results`
- `/admin/rounds/[id]/results/print`
- `RoundScoreInputForm`

## 적용 내용

1. 스코어 목록과 마이페이지 최근 기록의 보조 설명 문구를 제거했다.
2. 스코어가 없는 상태 문구를 짧게 통일했다.
3. 내 스코어 상세 화면의 반복 제목과 카드 여백을 줄였다.
4. 운영진 스코어 입력 화면의 상단 보조 라벨을 제거했다.
5. 스코어 입력 저장 버튼 문구를 더 짧게 조정했다.
6. 결과 화면의 `Top 3` 영문 표기를 `상위 3명`으로 변경했다.
7. 결과 화면의 공유/인쇄 영역에서 미리보기 설명 라벨을 제거했다.
8. 결과 계산 설명은 기본 노출 대신 `계산 기준` 접이식 영역으로 이동했다.
9. 결과가 없거나 참가자가 없는 빈 상태 문구를 짧게 정리했다.

## 확인 항목

- `/scores`에서 최근 기록 카드가 자연스럽게 보이는지 확인한다.
- `/scores/[roundId]`에서 내 스코어 카드가 간결하게 보이는지 확인한다.
- `/mypage` 최근 기록 영역에서 불필요한 설명이 사라졌는지 확인한다.
- `/admin/rounds/[id]/scores`에서 저장 버튼과 빈 상태가 자연스러운지 확인한다.
- `/admin/rounds/[id]/results`에서 `상위 3명`, `공유/인쇄`, `계산 기준`이 정상 표시되는지 확인한다.
- `/admin/rounds/[id]/results/print`에서 영문 `Top 3`가 남아 있지 않은지 확인한다.
