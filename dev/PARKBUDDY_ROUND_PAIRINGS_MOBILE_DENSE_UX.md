# PARKBUDDY_ROUND_PAIRINGS_MOBILE_DENSE_UX

## 목적

라운딩 관리 흐름에서 `/admin/rounds/[id]/pairings` 조 편성 화면의 모바일 사용성을 개선한다. 참가자 관리 목록 화면을 compact하게 정리한 다음 단계로, 조 편성 화면에서도 운영자가 적은 스크롤로 추천 조합, 경기 방식, 조별 배정을 확인하고 저장할 수 있게 한다.

## 변경 범위

- `src/app/(app)/admin/rounds/[id]/pairings/page.tsx`
- `src/components/admin/round-pairing-form.tsx`

## 적용 내용

### 조 편성 페이지

- 페이지 전체 여백을 축소했다.
- 제목 영역을 compact 카드형 헤더로 정리했다.
- 상단 이동 버튼 문구를 모바일 기준으로 짧게 줄였다.
- 저장/오류 메시지 카드 높이를 줄였다.
- 연결 일정 정보 표시 로직은 유지했다.

### 조 편성 폼

- 추천 조합 카드를 compact summary 카드로 정리했다.
- 추천 조 구성을 `1조 4명`, `2조 3명` 형태의 작은 chip으로 표시했다.
- 경기 형태/점수 계산 선택 영역의 높이와 여백을 줄였다.
- 수동 조 편성 목록을 모바일 1열, 태블릿 2열, 데스크톱 3열 카드 grid로 정리했다.
- 각 참가자 카드에서 이름, 핸디캡, 조 선택을 한 줄에 가깝게 배치했다.
- 조별 현재 배정 인원 chip을 표시해 저장 전 확인성을 높였다.
- 하단 sticky 저장 버튼은 유지했다.

## 비변경 범위

- 조 편성 저장 서버 액션은 변경하지 않았다.
- 조 편성 RPC/SQL은 변경하지 않았다.
- 참가자 선택, 스코어 입력, 결과 화면은 변경하지 않았다.
- 이미 연결된 일정 표시 로직은 유지했다.

## 확인 항목

1. `/admin/rounds/[id]/pairings` 접속
2. 추천 조합 카드가 compact하게 보이는지 확인
3. 경기 방식/점수 계산 select가 정상 동작하는지 확인
4. 참가자별 조 변경 시 조별 인원 chip이 갱신되는지 확인
5. 조 편성 저장 후 저장 완료 메시지가 보이는지 확인
6. 모바일에서 하단 저장 버튼이 정상 노출되는지 확인
