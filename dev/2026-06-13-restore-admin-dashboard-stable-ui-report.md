## 진행 내용

관리자 대시보드 UI가 의도치 않게 변경된 문제를 정리하기 위해, 대시보드 최초 추가 시점의 카드형 레이아웃을 기준으로 안정 UI를 재구성했다.

최근 관리자 작업 표시는 한글 작업명으로 유지했다.
