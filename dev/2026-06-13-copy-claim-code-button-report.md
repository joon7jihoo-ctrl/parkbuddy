# ParkBuddy 개발보고서 - 연결 코드 복사 버튼 추가

## 작성일

2026-06-13

## 진행 내용

운영진 회원 관리 화면에 연결 코드 복사 버튼을 추가했다.

## 변경 파일

```txt
src/components/copy-button.tsx
src/app/(app)/admin/members/page.tsx
dev/2026-06-13-copy-claim-code-button-report.md
```

## 구현 기능

회원 등록 또는 연결 코드 재발급 직후 표시되는 연결 코드 옆에 `코드 복사` 버튼을 추가했다.

운영진은 연결 코드를 직접 드래그하지 않고 버튼으로 클립보드에 복사할 수 있다.

## 구현 방식

클립보드 접근은 브라우저 API가 필요하므로 클라이언트 컴포넌트를 새로 추가했다.

```txt
src/components/copy-button.tsx
```

`navigator.clipboard.writeText()`를 사용하며, 복사 성공 후 버튼 문구를 잠시 `복사됨`으로 변경한다.

## 보안 평가

연결 코드는 기존처럼 등록/재발급 직후에만 표시된다.

이번 기능은 이미 화면에 표시된 값을 클립보드에 복사하는 보조 기능이다.

DB 저장 방식이나 연결 코드 검증 방식은 변경하지 않았다.

연결 코드는 여전히 DB에 평문 저장하지 않고 해시로만 저장된다.

## 완료 기준

```txt
1. npm run verify 통과
2. /admin/members 접속 가능
3. 회원 등록 후 연결 코드 옆에 코드 복사 버튼 표시
4. 버튼 클릭 시 코드 복사됨 문구 표시
5. 재발급 코드도 복사 가능
```
