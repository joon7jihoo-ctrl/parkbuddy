# ParkBuddy Schedule Kakao-like Poll UI

## 작업 일자

2026-06-16

## 대상 화면

- `/schedule`

## 변경 목적

일정 참석 카드가 사용자가 익숙한 메신저 투표 화면처럼 보이도록 레이아웃을 정리한다. 화면 내부에 `카카오톡 투표형` 같은 설명성 배지를 노출하지 않고, 일정 정보와 참석 선택만 남긴다.

## 변경 파일

- `src/app/(app)/schedule/page.tsx`
- `src/app/(app)/schedule/VoteButtons.tsx`
- `dev/PARKBUDDY_OPERATOR_DEV_STATUS.md`

## 변경 내용

- 일정 유형은 기존 내부값을 한국어 배지로만 표시한다.
  - `regular` 또는 기본값: `정기 라운딩`
  - `tournament`: `대회`
  - `casual`: `번개`
- `카카오톡 투표형` 설명 문구를 삭제했다.
- 참석 선택 UI를 2열 박스가 아니라 투표 항목 리스트 구조로 변경했다.
- 선택 항목은 체크 표시와 연한 노란색 배경으로 강조한다.
- 각 항목 오른쪽에는 `n명` 카운트를 표시한다.
- 참석 분류는 `참석`, `불참` 2개만 유지한다.
- 기존 `maybe` 값은 새 선택지로 노출하지 않고, 미선택처럼 안전 처리한다.
- 모바일 터치 영역은 최소 44px 이상을 유지한다.

## 검증

## 화면 확인 포인트

1. `/schedule` 카드에 `카카오톡 투표형` 문구가 보이지 않는다.
2. 일정 유형이 `정기 라운딩`, `대회`, `번개` 배지로 표시된다.
3. 참석 선택지가 `참석`, `불참` 두 줄 투표 항목처럼 표시된다.
4. 내가 선택한 항목에 체크 표시와 강조 배경이 적용된다.
5. 각 항목 오른쪽에 참석/불참 인원수가 `n명`으로 표시된다.
6. 모바일/태블릿/데스크탑에서 44px 이상 터치 영역과 2열 카드 배치가 유지된다.
