# 조 편성 후 점수 계산 방식 미지정 표시 오류 수정

## 작성일

2026-06-13

## 수정 배경

조 편성 저장 후 라운드 목록에서 점수 계산 방식이 미지정으로 표시되는 문제가 있었다.

원인은 매치 플레이 저장값이 match로 저장되는데 일부 화면의 한글 표시 함수가 match_play만 처리하거나, 조 편성 저장 후 라운드 메타 정보 반영이 확실하지 않은 흐름이 있었기 때문이다.

## 수정 내용

- 라운드 목록의 점수 계산 방식 표시에서 match 값을 매치 플레이로 표시하도록 보정
- 관련 표시 화면에서도 match와 match_play를 모두 매치 플레이로 표시하도록 보정
- 조 편성 저장 성공 후 rounds.game_type과 rounds.scoring_type을 한 번 더 갱신하도록 보정
- UI 구조와 참가자 선택 기능은 변경하지 않음

## 확인 항목

1. npm run verify 통과
2. 조 편성 화면에서 청백전 + 매치 플레이 저장
3. 라운드 목록에서 점수 계산 방식이 매치 플레이로 표시
4. 스트로크, 신페리오, 스테이블포드도 기존처럼 표시
