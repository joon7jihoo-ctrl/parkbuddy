# ParkBuddy 스코어 입력 카드 정렬 보정

## 작업 목적

스코어 입력 화면의 현재 2열/3열 카드 구성을 유지하면서, 카드 내부 입력 배치를 모든 화면 크기에서 동일하게 정리한다.

## 반영 내용

- 모바일, 태블릿, 데스크탑 모두 회원 카드 내부 배치를 동일하게 통일했다.
- 회원 정보 영역은 회원명, 핸디캡, 입력상태 순으로 세로 배치했다.
- 총 타수 입력 영역은 상대적으로 좁게, 스테이블포드 포인트 입력 영역은 상대적으로 넓게 설정했다.
- `스테이블포드 포인트` 라벨은 줄바꿈이 최소화되도록 `whitespace-nowrap`를 적용했다.
- 태블릿/데스크탑에서는 일자, 참가, 입력, 미입력만 보기 버튼이 4등분 균등 폭으로 배치되도록 수정했다.

## 확인 포인트

1. 모바일/태블릿/데스크탑 모두 카드 내부 순서가 동일한지 확인한다.
2. 회원명 아래에 핸디캡, 그 아래에 입력상태가 보이는지 확인한다.
3. 총 타수 입력창보다 스테이블포드 포인트 입력창이 더 넓게 잡히는지 확인한다.
4. 태블릿/데스크탑에서 일자, 참가, 입력, 미입력만 보기 버튼 폭이 균등한지 확인한다.
5. `npm run verify`가 통과하는지 확인한다.
