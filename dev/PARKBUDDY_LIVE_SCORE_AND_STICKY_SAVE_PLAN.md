# ParkBuddy 실시간 스코어 진행률 및 하단 고정 저장 버튼 개선 계획

## 작업명

`Add live score progress and sticky save actions`

## 목적

스코어 입력 중 완료 인원과 입력 완료율이 즉시 반영되도록 하고, 참가자 저장/조 편성 저장 버튼도 스코어 저장 버튼과 같은 하단 고정 CTA 패턴으로 통일한다.

## 반영 내용

- 스코어 입력 폼을 클라이언트 상태 기반 컴포넌트로 분리
- 총 타수 또는 스테이블포드 포인트 입력 시 완료율/완료 인원/미입력 인원 실시간 갱신
- 스코어 저장 버튼 문구도 실시간 완료 인원 기준으로 갱신
- 참가자 저장 버튼을 모바일 하단 고정 CTA로 이동
- 조 편성 저장 버튼을 모바일 하단 고정 CTA로 이동
- 데스크탑에서는 기존 흐름처럼 자연스럽게 문서 하단 버튼으로 표시

## 검증

- `npm run verify` 통과
- `/admin/rounds/[id]/scores`에서 입력 중 완료율과 저장 버튼 문구가 즉시 바뀌는지 확인
- `/admin/rounds/[id]/participants`에서 참가자 저장 버튼이 모바일 하단에 고정되는지 확인
- `/admin/rounds/[id]/pairings`에서 조 편성 저장 버튼이 모바일 하단에 고정되는지 확인
