# 스코어 저장 후 결과 보기 이동 검증 오류 정리

## 작성일

2026-06-13

## 처리 내용

스코어 저장 후 결과 보기 화면으로 이동하는 기능 적용 과정에서 생성된 임시 작업 폴더가 ESLint 검사 대상에 포함되어 `require()` 사용 오류가 발생했다.

앱 기능 파일은 유지하고, `.dev-temp` 임시 폴더만 삭제하도록 정리했다.

## 변경 범위

- `.dev-temp` 폴더 삭제
- 앱 코드 변경 없음

## 확인 기준

- `npm run verify` 통과
- 스코어 저장 후 결과 보기 화면으로 자동 이동
