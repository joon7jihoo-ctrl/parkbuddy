# ParkBuddy 회원 검색/필터/네비게이션 정리 계획

## 작업 목적

회원 관리와 회원 목록의 역할을 분리하고, 모바일에서 빠르게 검색/필터/주요 이동을 수행할 수 있도록 UI를 정리한다.

## 반영 내용

- 회원 검색은 한글 초성, 자소 분리, 부분 일치, 전화번호 숫자 검색, 키패드식 검색을 지원한다.
- 회원 관리 카드 필터는 검색어와 분리해 카드 선택 시 조건에 맞는 전체 목록을 보여준다.
- 검색창을 비우고 검색하면 전체 목록이 표시되므로 초기화 버튼을 제거한다.
- 회원 목록에서는 운영자 전용 관리/등록/연결 상태 정보를 제거한다.
- 회원 이름을 선택하면 상세/수정 화면으로 이동하고, 연락처 번호를 선택하면 전화 연결이 가능하다.
- 운영자는 하단 회원 메뉴가 회원 관리로 이동하고, 일반 회원은 회원 목록으로 이동한다.
- 운영자 하단 네비게이션에는 대시보드 메뉴를 추가한다.
- 운영자 대시보드는 통계 카드만 남겨 중복 메뉴를 제거한다.
- 라운드 목록은 상태별 보기 메뉴를 제거하고 전체/예정/완료/취소 카드 필터로 대체한다.
