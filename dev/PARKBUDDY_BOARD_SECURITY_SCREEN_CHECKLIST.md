# ParkBuddy 게시판 보안 화면 확인 체크리스트

작성일: 2026-06-15
목적: 게시판 비밀글/RLS/첨부 이미지 보안 패치 적용 후 브라우저에서 확인할 항목을 고정한다.

## 1. 사전 확인

- Supabase SQL Editor에서 `supabase/parkbuddy_board_private_security.sql` 실행 완료
- Supabase SQL Editor에서 `dev/PARKBUDDY_BOARD_SECURITY_SQL_CHECK.sql` 실행 결과 전체 PASS 확인
- 로컬에서 `npm run verify` 통과
- 확인 계정 준비
  - 운영진 계정 1개
  - 일반 회원 A 계정 1개
  - 일반 회원 B 계정 1개

## 2. 일반글 확인

1. 일반 회원 A로 로그인한다.
2. `/board/new`에서 자유게시글을 일반글로 작성한다.
3. `/board` 목록에 글이 보이는지 확인한다.
4. 일반 회원 B로 로그인해 같은 글이 목록과 상세에서 보이는지 확인한다.
5. 운영진 계정으로 로그인해 같은 글이 보이는지 확인한다.

기대 결과: 같은 클럽 회원과 운영진 모두 일반글을 볼 수 있다.

## 3. 비밀글 권한 확인

1. 일반 회원 A로 로그인한다.
2. `/board/new`에서 `비밀글로 작성`을 체크하고 자유게시글을 작성한다.
3. 일반 회원 A가 `/board` 목록과 상세에서 자기 비밀글을 볼 수 있는지 확인한다.
4. 운영진 계정으로 로그인해 해당 비밀글을 볼 수 있는지 확인한다.
5. 일반 회원 B로 로그인해 해당 비밀글이 `/board` 목록에 보이지 않는지 확인한다.
6. 일반 회원 B로 해당 비밀글 상세 URL을 직접 입력했을 때 조회되지 않는지 확인한다.

기대 결과: 비밀글은 작성자와 운영진만 볼 수 있고, 다른 일반 회원은 목록/직접 URL 접근 모두 차단된다.

## 4. 첨부 이미지 권한 확인

1. 일반 회원 A로 비밀글을 작성하면서 jpg/png/webp 중 하나를 첨부한다.
2. 작성자 계정에서 이미지가 보이는지 확인한다.
3. 운영진 계정에서 이미지가 보이는지 확인한다.
4. 일반 회원 B 계정에서 비밀글이 목록에 보이지 않는지 확인한다.
5. 이미지 URL을 알 수 있는 경우 일반 회원 B 세션에서 직접 열었을 때 접근되지 않는지 확인한다.

기대 결과: 비밀글 첨부 이미지는 작성자와 운영진에게만 노출된다.

## 5. 업로드 차단 확인

1. `.txt`, `.pdf`, `.gif` 등 허용되지 않은 파일 업로드를 시도한다.
2. 확장자만 `.jpg`로 바꾼 텍스트 파일 업로드를 시도한다.
3. 5MB 초과 이미지를 업로드 시도한다.

기대 결과: 허용 MIME, 실제 파일 시그니처, 용량 조건을 통과하지 못하면 저장되지 않는다.

## 6. XSS 입력 차단 확인

1. 제목에 `<script>alert(1)</script>` 입력을 시도한다.
2. 본문에 `<b>테스트</b>` 입력을 시도한다.

기대 결과: 꺾쇠 괄호 입력이 서버 검증에서 차단되어 저장되지 않는다.
