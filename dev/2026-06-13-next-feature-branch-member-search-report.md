# ParkBuddy 다음 기능 브랜치 준비 보고서

## 작성일
2026-06-13

## 목적
회원 수가 많아졌을 때 라운드 참가자 선택 화면에서 회원을 빠르게 찾을 수 있도록, 참가자 선택 화면 회원 검색 기능을 개발하기 위한 전용 Git 브랜치를 준비한다.

## 브랜치
feature/member-search-in-round-participants

## 작업 원칙
- 앱 코드는 아직 수정하지 않는다.
- 먼저 현재 상태가 Git 저장소인지 확인한다.
- 기존 변경사항이 있더라도 강제로 삭제하지 않는다.
- 브랜치를 생성하거나 이미 있으면 해당 브랜치로 이동한다.
- 이후 실제 기능 구현은 별도 단계에서 진행한다.

## 다음 단계
1. 라운드 참가자 선택 화면 파일 확인
2. 회원 목록 검색 입력창 추가
3. 이름/핸디캡/상태 기준 필터링
4. 선택된 참가자는 검색어와 관계없이 확인 가능하게 유지
5. npm run verify 실행
