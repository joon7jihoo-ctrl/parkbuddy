# ParkBuddy 회원 관리 검색/모바일 CTA 보정 계획

## 목적

회원 관리 화면에서 검색이 명확하게 동작하도록 보정하고, 모바일/데스크탑 모두에서 중복 카드와 불필요한 관리자 로그 영역을 줄인다.

## 반영 내용

- 회원 검색 form에 `method="get"`을 명시해 URL 검색 파라미터 기반으로 안정화한다.
- 검색 정규화에서 공백, 하이픈, 괄호, 점을 무시해 연락처 검색 성공률을 높인다.
- 회원 카드의 연락처/핸디캡/가입일/코드 만료 정보를 모든 화면 크기에서 2열 2줄로 유지한다.
- 회원 관리 화면의 `최근 관리자 작업 보기` 카드를 제거한다.
- 상단의 회원 등록 버튼을 제거하고, 회원 등록을 하단 고정 CTA로 통일한다.

## 검증 기준

- `/admin/members`에서 이름 검색이 동작한다.
- `/admin/members`에서 연락처 일부 또는 하이픈 없는 숫자 검색이 동작한다.
- 활성/비활성 필터와 검색어가 함께 유지된다.
- 데스크탑에서도 회원 정보 카드가 2열 2줄로 보인다.
- 최근 관리자 작업 카드가 보이지 않는다.
- 회원 등록 버튼이 하단 고정 CTA로 보인다.
- `npm run verify`가 통과한다.
