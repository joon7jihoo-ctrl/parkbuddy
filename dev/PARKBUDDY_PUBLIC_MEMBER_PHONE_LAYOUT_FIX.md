# ParkBuddy Public Member Phone Layout Fix

## 목적
회원 목록과 회원 상세 팝업에서 전화번호가 화면 폭에 따라 2줄 또는 3줄로 깨지는 문제를 줄인다.

## 변경 방향
- 가입일과 핸디캡을 상단 2칸에 배치한다.
- 연락처는 하단 전체 폭 1칸으로 이동한다.
- 연락처 값은 `whitespace-nowrap`와 `truncate`를 적용해 한 줄 표시를 우선한다.
- 연락처 클릭 시 전화 연결 동작은 유지한다.

## 확인 항목
- `/members` 회원 카드에서 전화번호가 한 줄로 보이는지 확인한다.
- 회원 상세 팝업에서도 연락처가 한 줄로 보이는지 확인한다.
- 연락처 클릭 시 전화 연결이 유지되는지 확인한다.
