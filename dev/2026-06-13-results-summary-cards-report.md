## 변경 파일

## 개선 내용

- 결과 화면 상단 요약 카드에 미입력 인원 표시를 추가했다.
- 결과 입력 현황 안내 영역을 추가했다.
- 입력 완료 인원과 미입력 인원을 결과 화면에서 바로 확인할 수 있게 했다.

## 안전 조치

- 기존 결과 페이지는 dev 폴더에 백업했다.
- 데이터베이스 변경은 없다.
- 기존 스코어 저장, 결과 정렬, 인쇄용 결과표 이동 기능은 유지한다.

## 확인 항목

- 입력 완료 인원 표시 확인
