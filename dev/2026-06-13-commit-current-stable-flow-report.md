## 목적
조 편성, 참가자 선택 보조 기능, 스코어 저장 후 결과 이동 흐름까지 확인된 현재 상태를 Git에 저장한다.

## 포함된 주요 작업
- 참가자 선택 검색/토글 기능 안정화
- 조 편성 저장 후 스코어 입력 화면 이동
- 스코어 저장 후 결과 보기 화면 이동
- verify 경고 정리 후 현재 상태 저장

## 실행 전제
- npm run verify가 통과해야 커밋을 진행한다.
- 현재 브랜치를 원격 저장소에 push한다.
